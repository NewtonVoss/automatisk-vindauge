## Bruk som utviding

Dette området kan bli lagt til som ei **utviding** i MakeCode.

* opne [https://makecode.microbit.org/](https://makecode.microbit.org/)
* klikk på **Nytt prosjekt**
* klikk på **Utvidingar** under girhjulmenyen
* søk etter **https://github.com/newtonvoss/automatisk-vindauge** og importer

## Rediger dette prosjektet ![Bygg statusmerke](https://github.com/newtonvoss/automatisk-vindauge/workflows/MakeCode/badge.svg)

For å redigere dette området i MakeCode.

* opne [https://makecode.microbit.org/](https://makecode.microbit.org/)
* klikk på ** Importer** og så på **Importer nettadresse**
* lim inn **https://github.com/newtonvoss/automatisk-vindauge** og klikk importer
